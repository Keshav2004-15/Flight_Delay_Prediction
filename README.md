# Flight_Delay_Prediction
The system finds the flight that have high chances of getting delayed. Once a potential delay is detected, the model supports decision-making by recommending lower-traffic time slots, reducing the likelihood of further delays and enhancing airport scheduling efficiency.

All codes are in single file 'mmdsendsemfinal.ipynb'
